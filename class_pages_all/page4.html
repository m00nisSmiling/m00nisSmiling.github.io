LEARNING-GUIDE FOR NEWBIES



  
For Newbies -

အခုမှ စပြီး zeroကနေ စပြီးလေ့လာမယ့် သင်တန်းသားတွေကို Newbie လို့ သတ်မှတ်ထားပါတယ်။Newbieတွေက ဘာလုပ်ရမလဲ?ဘယ်လိုလေ့လာရမလဲ?ဆိုတာတွေကို​သေချာ ​Guidelineချပေးပါမယ်။
ပထမဆုံးအနေနဲ့ Newbieတွေရဲ့ Roadmapကဘယ်လိုသွားရမလဲဆိုတော့ -

step1. websiteတွေရေးတဲ့အခါမှာ အသုံးပြုတဲ့Programming Languageတွေကို လေ့လာရပါမယ်။
(ကျွန်တော်တို့က Websiteတွေကို hackingပြုလုပ်မှာဖြစ်တဲ့အတွက် ပထမဦးစွာအနေနဲ့ websiteတွေဘယ်လို programmingတွေသုံးပြီး ဘယ်လိုအလုပ်လုပ်တာလဲဆိုတာ နားလည်အောင်လုပ်ရမယ့် stepပါ)

step2. kali linux ကို ကျွမ်းကျွမ်းကျင်ကျင် အသုံးပြုတတ်အောင် ကြိုးစားရပါမယ်။ (ကျွန်တော်တို့အတွက် Attacker machine or operating systemကို 
Kali linuxကို အသုံးပြုပြီး attack လုပ်ရမှာဖြစ်တဲ့အတွက်ကြောင့် ပါ)

step3. Tool setupတွေ attacking techinque တွေကို အခြေခံကနေ advance levelအထိ လေ့လာရပါမယ်။
(ဒါမှသာ real worldမှာ အေးအေးဆေးဆေး bugရှာနိုင်မှာမလို့ပါ)

step4. attacking techniqueတွေ အကုန်ပြီးလို့ real world ထိ အကုန်လေ့လာပြီးသွားတဲ့အခါ 
Pythonနဲ့ Phpကို လုပ်ကိုလုပ်ရပါမယ်။
(phpက ကမ္ဘာ့အသုံးများဆုံး web programming languageထဲမှာ ရေပန်းအစားဆုံး languageတစ်ခုဖြစ်တာမလို့ ကျွန်တော်တို့ ရင်ဆိုင်ရမယ့် web တော်တော်များများက php websiteတွေ ဖြစ်လာနိုင်ပါတယ်။)

Step5. ဒါတွေပြီးရင် နောက်ထပ် Free courseတွေ လေ့လာဖို့ ပေးထားတာရှိပါတယ် ဒါတွေထပ်လေ့လာရပါမယ်။


ဒီတော့ အထက်က Step5ခုမှာ
Step1မှာဆိုရင် ကျွန်တော် Classworkထဲက Essentials ထဲမှာပါတဲ့
Html playlistကို 15 to 20ရက်လောက်ထိ အချိန်ပေးပြီး html basicရအောင်လေ့လာရပါမယ်။
ဒါပြီးရင် cssအခြေခံကို 7 to 15 ရက်လောက်ထိ အချိန်ပေးပြီး အခြေခံ တစ်ခုရအောင် လေ့လာရပါမယ်။
ဒါပြီးရင် javascriptကို 1month to 2monthလောက်ထိ အချိန်ပေးပြီး အခြေခံတစ်ခုရှိအောင် လေ့လာရပါမယ်။
ဒါတွေပြီးရင်  Essentials ထဲက 1. Easy way to write deface page ခေါင်းစဥ်ကနေ အကုန်ကြည့်သွားရပါမယ်။
