Html,Css,Js စတဲ့ Markup and programming languagesတွေကို တတ်မြောက်ပြီးတဲ့သူ
အခြေခံရှိပြီး Attackingပိုင်းကို အခုမှ စပြီးလေ့လာမယ့်သူတွေကို kid လို့ ခေါ်မယ်။
သူတို့က ကျတော့ ပထမဆုံး -

Step1- သူတို့အနေနဲ့ ပထမဆုံး Phpကို လေ့လာဖို့လိုအပ်မယ်။
(ကမ္ဘာ့websiteတွေမှာအသုံးပြုတဲ့ languageတွေထဲမှာ ရေပန်းစားတဲ့languageဖြစ်တာကြောင့်ပါ၊ တစ်ချိန်ချိန် phpသုံးတဲ့ webတွေattackလုပ်တဲ့အခါမှာ နားလည်လွယ်စေဖို့)

Step2- Kali Linuxအကြောင်းကို နားလည်ဖို့ ကျွမ်းကျင်ဖို့ လိုအပ်မယ်။
(Kali linuxထဲမှာ Toolတွေ setupလုပ်ပြီး attacking marchineတစ်ခုအနေနဲ့ handleလုပ်ရမှာမလို့ ပါ)
Kali linux တင်ဖို့ Toolတွေ setupလုပ်ဖို့ကို Class workရဲ့ Ch1မှာ လေ့လာရပါမယ်။

Step3 - Attacking Concept And Techniqueတွေကို နားလည်ဖို့ ကျွမ်းကျင်ဖို့လိုမယ်။(ဒါတွေကျွမ်းကျင်မှသာ Real World Bug Huntလို့ရမှာမလို့ပါ)
ဒါတွေကို Classworkရဲ့ Ch2 Ch3 Ch4 Ch5မှာ သွားပြီး လေ့လာပြီး trainရပါမယ်။
ဒါတွေပြီးရင် Poc From Classဆိုတဲ့အောက်က Bug Bounty Poc Videoတွေ တစ်ခုချင်းစီကြည့်ပြီး နောက်ကွယ်က conceptကို ခန့်မှန်းနိုင်အောင်
impactတွေကို ခန့်မှန်းနိုင်အောင် လိုက်ပြီး ကြည့်ပြီး ကိုယ်တိုင် ရှာကြည့်ပါ။

Step4 - Pythonကိုလေ့လာဖို့လိုပါမယ်။
(pythonနဲ့ web exploit scriptတွေ automationတွေ အလွယ်တကူနဲ့ လုပ်ရမှာမလို့ပါ)
pythonကို classworkရဲ့ essentialsထဲမှာ လေ့လာဖို့ playlistချပေးထားပါတယ် အဲ့ကနေတစ်ဆင့်လေ့လာပြီး Automation and tool writingကိုတော့ classworkရဲ့ Ch6မှာ လေ့လာရပါမယ်။

Step5 - ဒါတွေပြီးရင်တော့ နောက်ဆုံးမှာ ချပေးထားတဲ့ Free Courseတွေကို တစ်ခုချင်းစီ ထပ် trainပြီး ကိုယ့်skillကိုယ် သွေးနေရပါမယ်။
ဒီcourseတွေကို ch7မှာ အကုန်ချပေးထားပါတယ်။
